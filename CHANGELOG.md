# CHANGELOG

## Додано
- Глобальний клас `.card` у `static/css/main.css` для уніфікації дизайну.
- Анімація появи (`fadeIn`) для карток для кращого UX.
- Розширений юридичний контент на сторінках Політики конфіденційності та Умов використання.

## Змінено
- Оптимізація для **Telegram Instant View**: додано Open Graph мета-теги та семантичну HTML5 розмітку (`<article>`, `<header>`, `<section>`).
- Оновлено дизайн юридичних сторінок: тепер вони базуються на системі карток.
- Видалено лоадер зі сторінок `/privacy` та `/terms` для миттєвого відображення контенту в Instant View.

## Безпека
- Оновлено мета-дані автора на `CraftLLC Internal`.
