**CraftLLC** — це сучасний веб-проєкт, створений для тих, хто цінує якісні рецепти, технології та безпеку. Наш сайт поєднує в собі кулінарний блог з інтерактивними елементами та корисними інструментами.

[**🌐 Перейти на сайт**](https://craftllc.pages.dev)

[![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/colored.png)](#--)

## ➤ ✨ Основні особливості

-   **📖 Кулінарна книга**: Велика база перевірених рецептів з можливістю ставити лайки та зберігати улюблені.
-   **🛡️ CraftAntivirus**: Інтегрований розділ для інноваційного антивірусу, що використовує VirusTotal та ШІ (Gemini) для аналізу файлів.
-   **🔑 Система акаунтів**: Персоналізація профілю, керування налаштуваннями та синхронізація вподобань.
-   **🎮 Cheat Codes**: Інтерактивна система чит-кодів для розблокування прихованих функцій або ефектів на сайті.
-   **❄️ Dynamic UI**: Сучасний дизайн з анімаціями (наприклад, ефект снігу), адаптивною версткою та темною темою.

[![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/colored.png)](#--)

## ➤ 🛠️ Технологічний стек

-   **Фронтенд**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
-   **Бекенд/API**: Cloudflare Workers (Edge Functions).
-   **База даних**: Cloudflare Workers KV (Key-Value Storage).
-   **Іконки**: FontAwesome 7.
-   **Хостинг**: Cloudflare Pages.
